How “upon” is used:


Structure:
Upon + noun/gerund (verb-ing), [main clause]


Meaning:
It signals that the main action happens right after the event in the “upon” phrase.



Examples:


Upon hearing the news, she burst into tears.
(As soon as she heard the news, she cried.)
